Open source project on GitHub

The online platforms are publicly accessible tools for providing and presenting data on national and global sustainability indicators. They have been developed in accordance with recognised international standards, particularly with regard to open-access data and open source software.

Sources

The Federal Statistical Office actively supports the development of national online platforms, particularly as an open source solution for presenting sustainability indicators. The United States and the United Kingdom are pioneers in this field. The project code for both the national and global platforms is publicly available in the following GitHub repositories:
  • National indicators
  • Global indicators
A universal version of the code for an online platform, developed by the United States, the United Kingdom, and the Center for Open Data Enterprise, is available online. The German online platforms for national and global sustainability indicators are based on this code. We recommend familiarising yourself with the platforms of the United States and the United Kingdom, as well as with the corresponding Open SDG project documentation, which includes technical instructions on how to create a copy of the Open SDG platform.
  • SDG online platform of the United States of America
  • SDG online platform of the United Kingdom
If you have any comments or feedback on the Open SDG project, or would like to participate in the Open SDG community, please contact Open SDG GitHub.

Software used

Back-end IT requirements:
  • GitHub: Website hosting, developed for project development using the Git version control system.
  • Jekyll: Static website generator based on Ruby.
Front-end IT requirements:
  • XHTML, CSS, JavaScript
  • Chartist: JavaScript library for creating charts.
  • Bootstrap: CSS framework
